## LUA 스크립트 설치
1. 관리자 모드 CMD 창에서 아래 명령어 수행

```
copy /Y ".\lp-acs-rot2prog.lua" "C:\Program Files\Wireshark\plugins\lp-acs-rot2prog.lua"
```

2. Wireshark 실행 시 자동 반영

3. Wireshark 실행 중이라면 Ctrl+Shift+L 입력 시 적용
